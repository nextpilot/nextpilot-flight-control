# Airships

<LinkedBadge type="warning" text="Experimental" url="../airframes/#experimental-vehicles"/>

:::warning
Support for airships is [experimental](../01.airframe_reference.md#experimental-vehicles).
Maintainer volunteers, [contribution]index.md) of new features, new frame configurations, or other improvements would all be very welcome!
:::

PX4 supports a single airship geometry with starboard, port and tail thrusters, and a thrust tilt server.
The frame configuration is shown in [Airframes Reference > Airship](../09.机型调参/01.airframe_reference.md#airship).
