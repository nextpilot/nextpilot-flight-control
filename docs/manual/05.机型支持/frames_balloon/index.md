# Balloons

<LinkedBadge type="warning" text="Experimental" url="../airframes/#experimental-vehicles"/>

:::warning
Support for balloons is [experimental](../01.airframe_reference.md#experimental-vehicles).
Maintainer volunteers, [contribution]index.md) of new features, new frame configurations, or other improvements would all be very welcome!
:::

PX4 supports a single balloon geometry with no flight controls.
The frame configuration is shown in [Airframes Reference > Balloon](../09.机型调参/01.airframe_reference.md#balloon).
