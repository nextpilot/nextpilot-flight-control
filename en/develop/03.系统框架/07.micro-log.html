    3. 03.зі»з»ҹжЎҶжһ¶

    зі»з»ҹж—Ҙеҝ—¶

    жіЁж„Ҹпјҡжң¬иҠӮиҜҙзҡ„ж—Ҙеҝ—жҳҜзі»з»ҹ**зі»з»ҹж—Ҙеҝ—**пјҢдёҚжҳҜ**йЈһиЎҢж•°жҚ®и®°еҪ•**пјҲиҜҘеҠҹиғҪеңЁloggerжЁЎеқ—дёӯе®һзҺ°пјүпјҢйҖҡдҝ—зҡ„и®ІпјҢе°ұжҳҜprintf()иҫ“еҮәеҲ°з»Ҳз«Ҝзҡ„ж–Үжң¬ж—Ҙеҝ—пјҒпјҒпјҒпјҒNextPilotеҹәдәҺдәҶRT-Threadзҡ„ulogз»„д»¶жқҘе®һзҺ°зі»з»ҹж—Ҙеҝ—зҡ„з®ЎзҗҶгҖӮ

    ж—Ҙеҝ—зҡ„е®ҡд№үпјҡж—Ҙеҝ—жҳҜе°ҶиҪҜд»¶иҝҗиЎҢзҡ„зҠ¶жҖҒгҖҒиҝҮзЁӢзӯүдҝЎжҒҜпјҢиҫ“еҮәеҲ°дёҚеҗҢзҡ„д»ӢиҙЁдёӯпјҲдҫӢеҰӮпјҡж–Үд»¶гҖҒжҺ§еҲ¶еҸ°гҖҒжҳҫзӨәеұҸзӯүпјүпјҢе№¶иҝӣиЎҢжҳҫзӨәе’ҢдҝқеӯҳгҖӮдёәиҪҜд»¶и°ғиҜ•гҖҒз»ҙжҠӨиҝҮзЁӢдёӯзҡ„й—®йўҳиҝҪжәҜгҖҒжҖ§иғҪеҲҶжһҗгҖҒзі»з»ҹзӣ‘жҺ§гҖҒж•…йҡңйў„иӯҰзӯүеҠҹиғҪпјҢжҸҗдҫӣеҸӮиҖғдҫқжҚ®гҖӮеҸҜд»ҘиҜҙпјҢж—Ҙеҝ—зҡ„дҪҝз”ЁпјҢеҮ д№ҺеҚ з”Ёзҡ„иҪҜд»¶з”ҹе‘Ҫе‘Ёжңҹзҡ„иҮіе°‘ 80% зҡ„ж—¶й—ҙгҖӮ

    ж—Ҙеҝ—зҡ„йҮҚиҰҒжҖ§пјҡеҜ№дәҺж“ҚдҪңзі»з»ҹиҖҢиЁҖпјҢз”ұдәҺе…¶иҪҜд»¶зҡ„еӨҚжқӮеәҰйқһеёёеӨ§пјҢеҚ•жӯҘи°ғиҜ•еңЁдёҖдәӣеңәжҷҜдёӢе№¶дёҚйҖӮеҗҲпјҢжүҖд»Ҙж—Ҙеҝ—з»„д»¶еңЁж“ҚдҪңзі»з»ҹдёҠеҮ д№ҺйғҪжҳҜж Үй…ҚгҖӮе®Ңе–„зҡ„ж—Ҙеҝ—зі»з»ҹд№ҹиғҪи®©ж“ҚдҪңзі»з»ҹзҡ„и°ғиҜ•дәӢеҚҠеҠҹеҖҚгҖӮ

    зі»з»ҹж—Ҙеҝ—еұӮж¬Ў¶

    еҮҪж•° дёӯж–ӯиҫ“еҮә ејӮжӯҘиҫ“еҮә з»Ҳз«ҜжҳҫзӨә зі»з»ҹж—Ҙеҝ— йЈһжҺ§ж—Ҙеҝ— ең°йқўз«ҷжҳҫзӨә
    rt_kprintf вҲҡ Г— вҲҡ Г— Г— Г—
    LOG_E/I/W/D/RAW вҲҡ вҲҡ вҲҡ вҲҡ вҲҡ Г—

    rt_kprintf¶

    rt_kprintf()жҳҜж ҮеҮҶеә“еҮҪж•°printf()зҡ„з®ҖеҢ–зүҲжң¬гҖӮеҸӘжҳҜеңЁFinshжҺ§еҲ¶еҸ°иҫ“еҮәпјҢдёҖиҲ¬жҳҜжү“еҚ°Finshз»Ҳз«Ҝзҡ„дәӨдә’жҢҮд»ӨпјҢз”ЁдәҺеөҢе…ҘејҸи°ғиҜ•гҖӮ

    еҰӮжһңдёҚжҳҜзү№ж®ҠйңҖжұӮпјҢе»әи®®дҪҝз”Ё rt_kprintf еҮҪж•°пјҢеӣ дёә printf жҳҜз”ұзј–иҜ‘е№іеҸ°еҶ…йғЁжҸҗдҫӣзҡ„пјҢе…¶з©әй—ҙеҚ з”ЁгҖҒд»ҘеҸҠеҶ…еӯҳдҪҝз”Ёжғ…еҶөжҲ‘д»¬ж— д»Һеҫ—зҹҘпјҢprintf еҮҪж•°иҰҒжҜ” rt_kprintf еҮҪж•°зҡ„ ROM еҚ з”ЁеӨ§еҫҲеӨҡгҖӮ

    ж— и®әжҳҜ rt_kprintf еҮҪж•°д»ҘеҸҠ printf еҮҪж•°йғҪжҳҜйқһзәҝзЁӢеҮҪж•°пјҢеңЁеӨҡзәҝзЁӢеҗҢж—¶дҪҝз”Ёзҡ„жғ…еҶөдёӢпјҢдјҡеҮәзҺ°дәӨеҸүжү“еҚ°зҡ„зҺ°иұЎпјҢиҜҘй—®йўҳжҳҜжӯЈеёёзҺ°иұЎпјҢеӣ дёәж №жҚ® C ж ҮеҮҶзҡ„иҰҒжұӮпјҢprintf еҮҪж•°е°ұжҳҜйқһзәҝзЁӢе®үе…Ёзҡ„гҖӮ

    еҺҹз”ҹ rt_kprintf еҮҪж•°жҳҜз»ҸиҝҮдјҳеҢ–зҡ„пјҢеҚ з”Ёз©әй—ҙиҰҒжҜ” printf еҮҪж•°е°ҸеҫҲеӨҡгҖӮдҪҶжҳҜ rt_kprintf еҮҪж•°е№¶дёҚж”ҜжҢҒжө®зӮ№зұ»еһӢзҡ„иҫ“еҮәгҖӮеӣ жӯӨпјҡ

    • еҰӮйңҖ rt_kprintf еҮҪж•°ж”ҜжҢҒжө®зӮ№зұ»еһӢзҡ„иҫ“еҮәпјҢеҸҜд»Ҙе®үиЈ… rt_vsnprintf_full иҪҜд»¶еҢ…гҖӮ
    • еҰӮйңҖ rt_kprintf еҮҪж•°ж”ҜжҢҒзәҝзЁӢе®үе…Ёиҫ“еҮәпјҢеҸҜд»Ҙе®үиЈ… rt_kprintf_threadsafe иҪҜд»¶еҢ…гҖӮ
    • дёҠиҝ°дёӨдёӘиҪҜд»¶еҢ…еҸҜд»ҘеҗҢж—¶е®үиЈ…дҪҝз”ЁпјҢд»ҘеҗҢж—¶и®© rt_kprintf ж”ҜжҢҒзәҝзЁӢе®үе…Ёе’Ңжө®зӮ№зұ»еһӢиҫ“еҮәзҡ„иғҪеҠӣгҖӮ

    LOG_xxx¶

    LOG_I/D/W/E()пјҢжҳҜRT-Thread ULOGе®ҳж–№жҸҗдҫӣзҡ„з»„д»¶гҖӮNextpilotйҖҡиҝҮж·»еҠ ULOGзҡ„еҗҺз«ҜеҠҹиғҪпјҢдҪҝеҫ—LOG_XXXе®Ҹе…·еӨҮеңЁз»Ҳз«Ҝиҫ“еҮәпјҲconsole backendпјүпјҢдҝқеӯҳеҲ°зі»з»ҹж—Ҙеҝ—ж–Үд»¶пјҲfile backendпјүпјҢдҝқеӯҳеҲ°йЈһжҺ§ж—Ҙеҝ—ж–Үд»¶пјҲuorb backendпјүгҖӮж”ҜжҢҒејӮжӯҘиҫ“еҮәпјҲдёҚеҚ з”Ёдё»зәҝзЁӢиө„жәҗпјүпјҢж”ҜжҢҒж ҮзӯҫгҖҒж—¶й—ҙжҲігҖҒеҪ©иүІзӯүгҖӮдёҖиҲ¬и®°еҪ•зЁӢеәҸиҝҗиЎҢзҡ„дәӢд»¶гҖҒжҸҗзӨәгҖҒиӯҰе‘ҠгҖҒй”ҷиҜҜзӯүпјҢз”ұдәҺи®°еҪ•еҲ°дәҶж–Үд»¶дёӯпјҢеӣ жӯӨеҸҜд»Ҙз”ЁжқҘиҫ…еҠ©йЈһиЎҢдәӢж•…еҲҶжһҗпјҢдәҶи§Јзі»з»ҹзҡ„иҝҗиЎҢиҝҮзЁӢгҖӮ

    // еңЁйңҖиҰҒдҪҝз”ЁLOG_XXXе®Ҹзҡ„C/C++ж–Үд»¶жңҖйЎ¶дёҠж·»еҠ 
#define LOG_TAG "mytag"  # иҝҷдёӘжҳҜж ҮзӯҫпјҢз”ЁжқҘеҢәеҲҶжҳҜж—Ҙеҝ—жқҘиҮӘе“ӘдёӘж–Үд»¶
#define LOG_LVL LOG_LVL_INFO # иҝҷдёӘжҳҜж—Ҙеҝ—иҫ“еҮәзә§еҲ«пјҢдҪҺдәҺиҝҷдёӘзә§еҲ«дёҚжү“еҚ°

// ж·»еҠ еӨҙж–Үд»¶пјҢLOG_TAGе’ҢLOG_LVLдёҖе®ҡиҰҒеңЁиҝҷдёӘд№ӢдёҠе®ҡд№ү
#include <rtdbg.h>

// do something
LOG_I("this is LOG_I"); // LOG_XXXдјҡиҮӘеҠЁж·»еҠ жҚўиЎҢ

    mavlink_log_xxx¶

    mavlink_log_xxx()пјҢжҳҜеҜ№LOG_XXXзҡ„е°ҒиЈ…пјҢе№¶иҪ¬дёәmavlinkж¶ҲжҒҜдёӢеҸ‘з»ҷең°йқўз«ҷпјҢеӣ жӯӨе…·еӨҮеңЁз»Ҳз«Ҝиҫ“еҮәгҖҒдҝқеӯҳеҲ°зі»з»ҹж—Ҙеҝ—ж–Үд»¶е’ҢйЈһжҺ§ж—Ҙеҝ—ж–Үд»¶гҖҒе№¶йҖҡиҝҮmavlinkеҸ‘йҖҒеҲ°ең°йқўз«ҷгҖӮдёҖиҲ¬жҳҜзү№еҲ«йҮҚиҰҒзҡ„дҝЎжҒҜпјҢеңЁйЈһиЎҢиҝҮзЁӢдёӯпјҢеҸҜд»ҘзӣҙжҺҘиҜӯйҹіжҸҗйҶ’з”ЁжҲ·гҖӮ

    ULOGз»„д»¶д»Ӣз»Қ¶

    ulog жҳҜдёҖдёӘйқһеёёз®ҖжҙҒгҖҒжҳ“з”Ёзҡ„ C/C++ ж—Ҙеҝ—з»„д»¶пјҢз¬¬дёҖдёӘеӯ—жҜҚ u д»ЈиЎЁ ОјпјҢеҚіеҫ®еһӢзҡ„ж„ҸжҖқгҖӮе®ғиғҪеҒҡеҲ°жңҖдҪҺ**ROM<1K, RAM<0.2K**зҡ„иө„жәҗеҚ з”ЁгҖӮulog дёҚд»…жңүе°Ҹе·§дҪ“з§ҜпјҢеҗҢж ·д№ҹжңүйқһеёёе…Ёйқўзҡ„еҠҹиғҪпјҢе…¶и®ҫи®ЎзҗҶеҝөеҸӮиҖғзҡ„жҳҜеҸҰеӨ–дёҖж¬ҫ C/C++ ејҖжәҗж—Ҙеҝ—еә“пјҡEasyLoggerпјҲз®Җз§° elogпјүпјҢе№¶еңЁеҠҹиғҪе’ҢжҖ§иғҪзӯүж–№йқўеҒҡдәҶйқһеёёеӨҡзҡ„ж”№иҝӣгҖӮдё»иҰҒзү№жҖ§еҰӮдёӢпјҡ

    • ж—Ҙеҝ—иҫ“еҮәзҡ„еҗҺз«ҜеӨҡж ·еҢ–пјҢеҸҜж”ҜжҢҒдҫӢеҰӮпјҡдёІеҸЈгҖҒзҪ‘з»ңпјҢж–Үд»¶гҖҒй—ӘеӯҳзӯүеҗҺз«ҜеҪўејҸгҖӮ

    • ж—Ҙеҝ—иҫ“еҮәиў«и®ҫи®ЎдёәзәҝзЁӢе®үе…Ёзҡ„ж–№ејҸпјҢе№¶ж”ҜжҢҒејӮжӯҘиҫ“еҮәжЁЎејҸгҖӮ

    • ж—Ҙеҝ—зі»з»ҹй«ҳеҸҜйқ пјҢеңЁдёӯж–ӯ ISR гҖҒHardfault зӯүеӨҚжқӮзҺҜеўғдёӢдҫқж—§еҸҜз”ЁгҖӮ

    • ж—Ҙеҝ—ж”ҜжҢҒиҝҗиЎҢжңҹ / зј–иҜ‘жңҹи®ҫзҪ®иҫ“еҮәзә§еҲ«гҖӮ

    • ж—Ҙеҝ—еҶ…е®№ж”ҜжҢҒжҢүе…ій”®иҜҚеҸҠж Үзӯҫж–№ејҸиҝӣиЎҢе…ЁеұҖиҝҮж»ӨгҖӮ

    • API е’Ңж—Ҙеҝ—ж јејҸеҸҜе…је®№ linux syslogгҖӮ

    • ж”ҜжҢҒд»Ҙ hex ж јејҸ dump и°ғиҜ•ж•°жҚ®еҲ°ж—Ҙеҝ—дёӯгҖӮ

    • е…је®№ rtdbg пјҲRTT ж—©жңҹзҡ„ж—Ҙеҝ—еӨҙж–Үд»¶пјүеҸҠ EasyLogger зҡ„ж—Ҙеҝ—иҫ“еҮә APIгҖӮ

    дёӢеӣҫдёә ulog ж—Ҙеҝ—з»„д»¶жһ¶жһ„еӣҫпјҡ

    • еүҚз«ҜпјҡиҜҘеұӮдҪңдёәзҰ»еә”з”ЁжңҖиҝ‘зҡ„дёҖеұӮпјҢз»ҷз”ЁжҲ·жҸҗдҫӣдәҶ syslog еҸҠ LOG_X дёӨзұ» API жҺҘеҸЈпјҢж–№дҫҝз”ЁжҲ·еңЁдёҚеҗҢзҡ„еңәжҷҜдёӯдҪҝз”ЁгҖӮ

    • ж ёеҝғпјҡдёӯй—ҙж ёеҝғеұӮзҡ„дё»иҰҒе·ҘдҪңжҳҜе°ҶдёҠеұӮдј йҖ’иҝҮжқҘзҡ„ж—Ҙеҝ—пјҢжҢүз…§дёҚеҗҢзҡ„й…ҚзҪ®иҰҒжұӮиҝӣиЎҢж јејҸеҢ–дёҺиҝҮж»Өз„¶еҗҺз”ҹжҲҗж—Ҙеҝ—её§пјҢжңҖз»ҲйҖҡиҝҮдёҚеҗҢзҡ„иҫ“еҮәжЁЎеқ—пјҢиҫ“еҮәеҲ°жңҖеә•еұӮзҡ„еҗҺз«Ҝи®ҫеӨҮдёҠгҖӮ

    • еҗҺз«ҜпјҡжҺҘж”¶еҲ°ж ёеҝғеұӮеҸ‘жқҘзҡ„ж—Ҙеҝ—её§еҗҺпјҢе°Ҷж—Ҙеҝ—иҫ“еҮәеҲ°е·Із»ҸжіЁеҶҢзҡ„ж—Ҙеҝ—еҗҺз«Ҝи®ҫеӨҮдёҠпјҢдҫӢеҰӮпјҡж–Үд»¶гҖҒжҺ§еҲ¶еҸ°гҖҒж—Ҙеҝ—жңҚеҠЎеҷЁзӯүзӯүгҖӮ

    ж—Ҙеҝ—зә§еҲ«¶

    ж—Ҙеҝ—зә§еҲ«д»ЈиЎЁдәҶж—Ҙеҝ—зҡ„йҮҚиҰҒжҖ§пјҢеңЁ ulog дёӯз”ұй«ҳеҲ°дҪҺпјҢжңүеҰӮдёӢеҮ дёӘж—Ҙеҝ—зә§еҲ«пјҡ

    зә§еҲ« еҗҚз§° жҸҸиҝ°
    LOG_LVL_ASSERT ж–ӯиЁҖ еҸ‘з”ҹж— жі•еӨ„зҗҶгҖҒиҮҙе‘ҪжҖ§зҡ„зҡ„й”ҷиҜҜпјҢд»ҘиҮідәҺзі»з»ҹж— жі•з»§з»ӯиҝҗиЎҢзҡ„ж–ӯиЁҖж—Ҙеҝ—
    LOG_LVL_ERROR й”ҷиҜҜ еҸ‘з”ҹдёҘйҮҚзҡ„гҖҒ**дёҚеҸҜдҝ®еӨҚ**зҡ„й”ҷиҜҜж—¶иҫ“еҮәзҡ„ж—Ҙеҝ—еұһдәҺй”ҷиҜҜзә§еҲ«ж—Ҙеҝ—
    LOG_LVL_WARNING иӯҰе‘Ҡ еҮәзҺ°дёҖдәӣдёҚеӨӘйҮҚиҰҒзҡ„гҖҒе…·жңү**еҸҜдҝ®еӨҚжҖ§**зҡ„й”ҷиҜҜж—¶пјҢдјҡиҫ“еҮәиҝҷдәӣиӯҰе‘Ҡж—Ҙеҝ—
    LOG_LVL_INFO дҝЎжҒҜ з»ҷжң¬жЁЎеқ—дёҠеұӮдҪҝз”Ёдәәе‘ҳжҹҘзңӢзҡ„йҮҚиҰҒжҸҗзӨәдҝЎжҒҜж—Ҙеҝ—пјҢдҫӢеҰӮпјҡеҲқе§ӢеҢ–жҲҗеҠҹпјҢеҪ“еүҚе·ҘдҪңзҠ¶жҖҒзӯүгҖӮиҜҘзә§еҲ«ж—Ҙеҝ—дёҖиҲ¬еңЁйҮҸдә§ж—¶дҫқж—§**дҝқз•ҷ**
    LOG_LVL_DBG и°ғиҜ• з»ҷжң¬жЁЎеқ—ејҖеҸ‘дәәе‘ҳжҹҘзңӢзҡ„и°ғиҜ•ж—Ҙеҝ—пјҢиҜҘзә§еҲ«ж—Ҙеҝ—дёҖиҲ¬еңЁйҮҸдә§ж—¶**е…ій—ӯ**

    еңЁ ulog дёӯж—Ҙеҝ—зә§еҲ«иҝҳжңүеҰӮдёӢеҲҶзұ»пјҡ

    • йқҷжҖҒзә§еҲ«дёҺеҠЁжҖҒзә§еҲ«пјҡжҢүз…§ж—Ҙеҝ—жҳҜеҗҰеҸҜд»ҘеңЁиҝҗиЎҢйҳ¶ж®өдҝ®ж”№иҝӣиЎҢеҲҶзұ»гҖӮеҸҜеңЁиҝҗиЎҢйҳ¶ж®өдҝ®ж”№зҡ„з§°д№ӢдёәеҠЁжҖҒзә§еҲ«пјҢеҸӘиғҪеңЁ**зј–иҜ‘йҳ¶ж®ө**дҝ®ж”№зҡ„з§°д№ӢдёәйқҷжҖҒзә§еҲ«гҖӮжҜ”йқҷжҖҒзә§еҲ«дҪҺзҡ„ж—Ҙеҝ—пјҲиҝҷйҮҢзү№жҢҮдҪҝз”Ё LOG_X API зҡ„ж—Ҙеҝ—пјүе°ҶдёҚдјҡиў«зј–иҜ‘еҲ° ROM дёӯпјҢжңҖз»Ҳд№ҹдёҚдјҡиҫ“еҮәгҖҒжҳҫзӨәеҮәжқҘгҖӮиҖҢеҠЁжҖҒзә§еҲ«еҸҜд»Ҙз®ЎжҺ§зҡ„жҳҜй«ҳдәҺжҲ–зӯүдәҺйқҷжҖҒзә§еҲ«зҡ„ж—Ҙеҝ—гҖӮеңЁ ulog иҝҗиЎҢж—¶пјҢжҜ”еҠЁжҖҒзә§еҲ«дҪҺзҡ„ж—Ҙеҝ—дјҡиў«иҝҮж»ӨжҺүгҖӮ

    • е…ЁеұҖзә§еҲ«дёҺжЁЎеқ—зә§еҲ«пјҡжҢүз…§дҪңз”ЁеҹҹиҝӣиЎҢзҡ„еҲҶзұ»гҖӮеңЁ ulog дёӯжҜҸдёӘж–Үд»¶пјҲжЁЎеқ—пјүд№ҹеҸҜд»Ҙи®ҫе®ҡзӢ¬з«Ӣзҡ„ж—Ҙеҝ—зә§еҲ«гҖӮе…ЁеұҖзә§еҲ«дҪңз”ЁеҹҹеӨ§дәҺжЁЎеқ—зә§еҲ«пјҢд№ҹе°ұжҳҜжЁЎеқ—зә§еҲ«еҸӘиғҪз®ЎжҺ§йӮЈдәӣй«ҳдәҺжҲ–зӯүдәҺе…ЁеұҖзә§еҲ«зҡ„жЁЎеқ—ж—Ҙеҝ—гҖӮ

    з»јеҗҲдёҠйқўеҲҶзұ»еҸҜд»ҘзңӢеҮәпјҢеңЁ ulog еҸҜд»ҘйҖҡиҝҮд»ҘдёӢ 4 дёӘж–№йқўжқҘи®ҫе®ҡж—Ҙеҝ—зҡ„иҫ“еҮәзә§еҲ«пјҡ

    • **е…ЁеұҖйқҷжҖҒ**ж—Ҙеҝ—зә§еҲ«пјҡеңЁ menuconfig дёӯй…ҚзҪ®пјҢеҜ№еә” ULOG_OUTPUT_LVL е®ҸгҖӮ

    • **е…ЁеұҖеҠЁжҖҒ**ж—Ҙеҝ—зә§еҲ«пјҡдҪҝз”Ё void ulog_global_filter_lvl_set(rt_uint32_t level) еҮҪж•°жқҘи®ҫе®ҡгҖӮ

    • **жЁЎеқ—йқҷжҖҒ**ж—Ҙеҝ—зә§еҲ«пјҡеңЁжЁЎеқ—пјҲж–Үд»¶пјүеҶ…е®ҡд№ү LOG_LVL е®ҸпјҢдёҺж—Ҙеҝ—ж Үзӯҫе®Ҹ LOG_TAG е®ҡд№үж–№ејҸзұ»дјјгҖӮ

    • **жЁЎеқ—еҠЁжҖҒ**ж—Ҙеҝ—зә§еҲ«пјҡдҪҝз”Ё int ulog_tag_lvl_filter_set(const char *tag, rt_uint32_t level) еҮҪж•°жқҘи®ҫе®ҡгҖӮ

    е®ғд»¬зҡ„дҪңз”ЁиҢғеӣҙе…ізі»дёәпјҡе…ЁеұҖйқҷжҖҒ>е…ЁеұҖеҠЁжҖҒ>жЁЎеқ—йқҷжҖҒ>жЁЎеқ—еҠЁжҖҒгҖӮ

    ж—Ҙеҝ—ж Үзӯҫ¶

    з”ұдәҺж—Ҙеҝ—иҫ“еҮәйҮҸзҡ„дёҚж–ӯеўһеӨ§пјҢдёәдәҶйҒҝе…Қж—Ҙеҝ—иў«жқӮд№ұж— з« зҡ„иҫ“еҮәеҮәжқҘпјҢе°ұйңҖиҰҒдҪҝз”Ёж ҮзӯҫпјҲtagпјүз»ҷжҜҸжқЎж—Ҙеҝ—иҝӣиЎҢеҲҶзұ»гҖӮж Үзӯҫзҡ„е®ҡд№үжҳҜжҢүз…§**жЁЎеқ—еҢ–**зҡ„ж–№ејҸпјҢдҫӢеҰӮпјҡWi-Fi з»„д»¶еҢ…жӢ¬и®ҫеӨҮй©ұеҠЁпјҲwifi_driverпјүгҖҒи®ҫеӨҮз®ЎзҗҶпјҲwifi_mgntпјүзӯүжЁЎеқ—пјҢеҲҷ Wi-Fi з»„д»¶еҶ…йғЁжЁЎеқ—еҸҜд»ҘдҪҝз”Ё wifi.driverгҖҒwifi.mgnt зӯүдҪңдёәж ҮзӯҫпјҢиҝӣиЎҢж—Ҙеҝ—зҡ„еҲҶзұ»иҫ“еҮәгҖӮ

    жҜҸжқЎж—Ҙеҝ—зҡ„ж ҮзӯҫеұһжҖ§д№ҹеҸҜд»Ҙиў«иҫ“еҮәе№¶жҳҫзӨәеҮәжқҘпјҢеҗҢж—¶ ulog иҝҳеҸҜд»Ҙи®ҫзҪ®жҜҸдёӘж ҮзӯҫпјҲжЁЎеқ—пјүеҜ№еә”ж—Ҙеҝ—зҡ„иҫ“еҮәзә§еҲ«пјҢеҪ“еүҚдёҚйҮҚиҰҒжЁЎеқ—зҡ„ж—Ҙеҝ—еҸҜд»ҘйҖүжӢ©жҖ§е…ій—ӯпјҢдёҚд»…йҷҚдҪҺ ROM иө„жәҗпјҢиҝҳиғҪеё®еҠ©ејҖеҸ‘иҖ…иҝҮж»Өж— е…іж—Ҙеҝ—гҖӮ

    еҸӮи§Ғ rt-thread\examples\ulog_example.c ulog дҫӢзЁӢж–Үд»¶пјҢеңЁж–Үд»¶йЎ¶йғЁжңүе®ҡд№ү LOG_TAG е®Ҹпјҡ

    #define LOG_TAG     "example"     // иҜҘжЁЎеқ—еҜ№еә”зҡ„ж ҮзӯҫгҖӮдёҚе®ҡд№үж—¶пјҢй»ҳи®ӨпјҡNO_TAG
#define LOG_LVL     LOG_LVL_DBG   // иҜҘжЁЎеқ—еҜ№еә”зҡ„ж—Ҙеҝ—иҫ“еҮәзә§еҲ«гҖӮдёҚе®ҡд№үж—¶пјҢй»ҳи®Өпјҡи°ғиҜ•зә§еҲ«
#include <ulog.h>                 // еҝ…йЎ»еңЁ LOG_TAG дёҺ LOG_LVL дёӢйқў

    йңҖиҰҒжіЁж„Ҹзҡ„пјҢе®ҡд№үж—Ҙеҝ—ж Үзӯҫеҝ…йЎ»дҪҚдәҺ #include <ulog.h> зҡ„дёҠж–№пјҢеҗҰеҲҷдјҡдҪҝз”Ёй»ҳи®Өзҡ„ NO_TAGпјҲдёҚжҺЁиҚҗе®ҡд№үеңЁеӨҙж–Үд»¶дёӯе®ҡд№үиҝҷдәӣе®ҸпјүгҖӮ

    ж—Ҙеҝ—ж Үзӯҫзҡ„дҪңз”ЁеҹҹжҳҜеҪ“еүҚжәҗз Ғж–Үд»¶пјҢйЎ№зӣ®жәҗд»Јз ҒйҖҡеёёд№ҹдјҡжҢүз…§жЁЎеқ—иҝӣиЎҢж–Үд»¶еҲҶзұ»гҖӮжүҖд»ҘеңЁе®ҡд№үж Үзӯҫж—¶пјҢеҸҜд»ҘжҢҮе®ҡжЁЎеқ—еҗҚгҖҒеӯҗжЁЎеқ—еҗҚдҪңдёәж ҮзӯҫеҗҚз§°пјҢиҝҷж ·дёҚд»…еңЁж—Ҙеҝ—иҫ“еҮәжҳҫзӨәж—¶жё…жҷ°зӣҙи§ӮпјҢд№ҹиғҪж–№дҫҝеҗҺз»ӯжҢүж Үзӯҫж–№ејҸеҠЁжҖҒи°ғж•ҙзә§еҲ«жҲ–иҝҮж»ӨгҖӮ

    иҫ“еҮә API¶

    ulog дё»иҰҒжңүдёӨз§Қж—Ҙеҝ—иҫ“еҮәе®Ҹ APIпјҢжәҗд»Јз Ғдёӯе®ҡд№үеҰӮдёӢжүҖзӨәпјҡ

    #define LOG_E(...)                           ulog_e(LOG_TAG, __VA_ARGS__)
#define LOG_W(...)                           ulog_w(LOG_TAG, __VA_ARGS__)
#define LOG_I(...)                           ulog_i(LOG_TAG, __VA_ARGS__)
#define LOG_D(...)                           ulog_d(LOG_TAG, __VA_ARGS__)
#define LOG_RAW(...)                         ulog_raw(__VA_ARGS__)
#define LOG_HEX(name, width, buf, size)      ulog_hex(name, width, buf, size)

    • е®Ҹ LOG_X(...)пјҡX еҜ№еә”зҡ„жҳҜдёҚеҗҢзә§еҲ«зҡ„з¬¬дёҖдёӘеӯ—жҜҚеӨ§еҶҷгҖӮеҸӮж•° ... дёәж—Ҙеҝ—еҶ…е®№пјҢж јејҸдёҺ printf дёҖиҮҙгҖӮиҝҷз§Қж–№ејҸжҳҜйҰ–йҖүпјҢдёҖж–№йқўеӣ дёәе…¶ API ж јејҸз®ҖеҚ•пјҢе…ҘеҸӮеҸӘжңүдёҖдёӘеҚіж—Ҙеҝ—дҝЎжҒҜпјҢеҶҚиҖ…иҝҳж”ҜжҢҒжҢүжЁЎеқ—йқҷжҖҒж—Ҙеҝ—зә§еҲ«иҝҮж»ӨгҖӮ

    • е®Ҹ ulog_x(LOG_TAG, __VA_ARGS__)пјҡx еҜ№еә”зҡ„жҳҜдёҚеҗҢзә§еҲ«зҡ„з®ҖеҶҷгҖӮеҸӮж•° LOG_TAG дёәж—Ҙеҝ—ж ҮзӯҫпјҢеҸӮж•° ... дёәж—Ҙеҝ—еҶ…е®№пјҢж јејҸдёҺ printf дёҖиҮҙгҖӮиҝҷдёӘ API йҖӮз”ЁдәҺеңЁдёҖдёӘж–Үд»¶дёӯдҪҝз”ЁдёҚеҗҢ tag иҫ“еҮәж—Ҙеҝ—зҡ„жғ…еҶөгҖӮ

    API жҸҸиҝ°
    LOG_E(...) й”ҷиҜҜзә§еҲ«ж—Ҙеҝ—
    LOG_W(...) й”ҷиҜҜзә§еҲ«ж—Ҙеҝ—
    LOG_I(...) жҸҗзӨәзә§еҲ«ж—Ҙеҝ—
    LOG_D(...) и°ғиҜ•зә§еҲ«ж—Ҙеҝ—
    LOG_RAW(...) иҫ“еҮә raw ж—Ҙеҝ—
    LOG_HEX(name, width, buf, size) иҫ“еҮә 16 иҝӣеҲ¶ж јејҸж•°жҚ®еҲ°ж—Ҙеҝ—

    LOG_X еҸҠ ulog_x иҝҷзұ» API иҫ“еҮәйғҪжҳҜеёҰж јејҸж—Ҙеҝ—пјҢжңүдәӣж—¶еҖҷйңҖиҰҒиҫ“еҮәдёҚеёҰд»»дҪ•ж јејҸзҡ„ж—Ҙеҝ—ж—¶пјҢеҸҜд»ҘдҪҝз”Ё LOG_RAW жҲ– ulog_raw() гҖӮдҫӢеҰӮпјҡ

    LOG_RAW("\r");
ulog_raw("\033[2A");

    д»Ҙ 16 иҝӣеҲ¶ hex ж јејҸ dump ж•°жҚ®еҲ°ж—Ҙеҝ—дёӯеҸҜдҪҝз”ЁеҸҜд»ҘдҪҝз”Ё LOG_HEX() жҲ– ulog_hex гҖӮеҮҪж•°еҸӮж•°еҸҠжҸҸиҝ°еҰӮдёӢжүҖзӨәпјҡ

    еҸӮж•° жҸҸиҝ°
    tag ж—Ҙеҝ—ж Үзӯҫ
    width дёҖиЎҢ hex еҶ…е®№зҡ„е®ҪеәҰпјҲж•°йҮҸпјү
    buf еҫ…иҫ“еҮәзҡ„ж•°жҚ®еҶ…е®№
    size ж•°жҚ®еӨ§е°Ҹ

    hexdump ж—Ҙеҝ—дёә DEBUG зә§еҲ«пјҢж”ҜжҢҒиҝҗиЎҢжңҹзҡ„зә§еҲ«иҝҮж»ӨпјҢhexdump ж—Ҙеҝ—еҜ№еә”зҡ„ tag пјҢж”ҜжҢҒиҝҗиЎҢжңҹзҡ„ж ҮзӯҫиҝҮж»ӨгҖӮ

    ulog д№ҹжҸҗдҫӣйҮҢж–ӯиЁҖ API пјҡASSERT(иЎЁиҫҫејҸ) пјҢеҪ“ж–ӯиЁҖи§ҰеҸ‘ж—¶пјҢзі»з»ҹдјҡеҒңжӯўиҝҗиЎҢпјҢеҶ…йғЁд№ҹдјҡжү§иЎҢ ulog_flush() пјҢжүҖжңүж—Ҙеҝ—еҗҺз«Ҝе°Ҷжү§иЎҢ flush гҖӮеҰӮжһңејҖеҗҜдәҶејӮжӯҘжЁЎејҸпјҢзј“еҶІеҢәдёӯжүҖжңүзҡ„ж—Ҙеҝ—д№ҹе°Ҷиў« flush гҖӮж–ӯиЁҖзҡ„дҪҝз”ЁзӨәдҫӢеҰӮдёӢпјҡ
